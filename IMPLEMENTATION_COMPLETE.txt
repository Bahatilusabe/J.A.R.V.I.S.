API ENDPOINTS
================================================================================

ANALYSIS ENDPOINTS:
  POST /api/threat-intelligence/analyze
       Request: {text, source_url?, marketplace?}
       Response: ThreatSignal (full threat info)

  POST /api/threat-intelligence/marketplace
       Request: {title, description, marketplace, price?, vendor?}
       Response: ThreatSignal (classified listing)

  POST /api/threat-intelligence/leak-dump
       Request: {dump_name, entries_sample}
       Response: ThreatSignal (breach classification)

QUERY ENDPOINTS:
  GET /api/threat-intelligence/threats
       Response: List[ThreatSignal] (paginated)

  GET /api/threat-intelligence/threats/{id}
       Response: ThreatSignal (specific threat)

  GET /api/threat-intelligence/search?keyword=X&limit=20
       Response: List[ThreatSignal] (search results)

  GET /api/threat-intelligence/top?limit=10
       Response: List[ThreatSignal] (highest confidence)

REPORTING ENDPOINTS:
  GET /api/threat-intelligence/summary
       Response: ThreatSummaryResponse (statistics)

  GET /api/threat-intelligence/correlations?limit=20
       Response: List[CorrelationResponse] (relationships)

  GET /api/threat-intelligence/indicators
       Response: {cves, malware, threat_actors, domains}

MANAGEMENT ENDPOINTS:
  POST /api/threat-intelligence/correlate
       Request: {signal_id_1, signal_id_2}
       Response: CorrelationResponse (relationship)

  POST /api/threat-intelligence/alert
       Request: {keyword}
       Response: KeywordAlertResponse (matches)

INTEGRATION ENDPOINTS:
  POST /api/threat-intelligence/scrape
       Request: {urls, marketplace?}
       Response: Scrape results with signals

  GET /api/threat-intelligence/health
       Response: HealthResponse (service status)
